## 最近最少使用算法(Least Recently Used LRU)

### 算法介绍

### LRU原理

假设页号是7 0 1 2 0 3 0 4，分配三个物理块，一开始1-3趟，是把物理块填满，也就是：

+ 第一趟 7
+ 第二趟0 7
+ 第三趟1 0 7

到了第四趟，发生缺页，则把最近最久没有被访问的页给替换掉，最近没有使用的是7，所以第四趟结果如下：

+ 2 1 0
